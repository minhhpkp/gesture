# Gesture

A video communication platform designed to reduce the language barrier between the Deaf and hard-of-hearing community and the hearing world.

## How to run

1. Install and start LiveKit. Refer to [LiveKit guide](https://github.com/livekit/livekit?tab=readme-ov-file#getting-started).

2. Create a Python virtual environment or a Conda environment. For example:

```bash
conda create --name gesture
```

3. Refer to [app](src/app/README.md), [inference_server](src/inference_server/README.md), [connector](src/connector/README.md) and [auth](src/auth/README.md) for specfic instructions.
