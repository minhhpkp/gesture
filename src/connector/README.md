# Connector

Forward the signer's video to the inference server and broadcast the recognition results.

## How to run

1. Activate your Python virtual environment or Conda environment. For example:

```bash
conda activate gesture
```

2. Install dependencies:

```bash
pip install -r requirements.txt"
```

3. Create an `.env` file with fields as listed in [example.env](example.env)
4. Run:

```bash
python main.py
```
